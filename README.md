<h2>Descrição do Projeto</h2>
<p>
  Projeto em desenvolvimento para disciplina Sistemas Distribuídos, do curso de Engenharia de Software da Universidade Tecnológica Federal do Paraná. O SirSisDistribuidos é um sistema que permite a execução dos modelos SIR e SIS de epidemiologia matemática, cujo objetivo é simular a propagação de doenças infecciosas. <br>
  O sistema será implementado em 3 modos: sequencial, paralelo e distribuído.
</p>

<h2>Funcionalidades</h2>

- `Simulação do Modelo SIR e SIS`: Executa a simulação do modelos com base nos parâmetros informados pelo usuário.
- `Simulação em Gráfico`: Expõe um gráfico mostrando todo o comportamento dos agentes patogênicos ao longo do período informado.
- `Simulação em modo Paralelo`: Utiliza threads para paralelizar a simulação.
- `Simulação em modo Distribuído`: Implementa a arquitetura Cliente-Servidor com RMI para distribuir o sistema.

<h2>Tecnologias Utilizadas</h2>

- `JAVA`
- `Maven`
- `RMI`
